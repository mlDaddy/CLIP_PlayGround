# CLIP PlayGround for Testing CLIP Capabilities
This repo is used to demonstrate the powerful AI capabilities of OpenAI's CLIP Model.

# Setup Environment
```
conda create -n clip python=3.10
pip install -r requirements.txt
```

# Run Inference
```
python main.py
```
